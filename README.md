steps-sms-text-message
======================

SMS Text Message Sender using https://www.twilio.com/

To use their service you have to register first.

# Input Environment Variables
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN
- TWILIO_TO_NUMBER
- TWILIO_FROM_NUMBER
- TWILIO_SMS_MESSAGE
- TWILIO_SMS_MEDIA
